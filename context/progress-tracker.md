## Architecture Decisions

- Using shadcn/ui base-nova style with Tailwind v4 and CSS variables for theming
- Arabic font: Cairo (via next/font), English font: Inter (via next/font), mapped to `--font-sans` in globals.css
- RTL support via `dir` attribute on `<html>`, prepared but no translations yet
- Primary brand color uses oklch blue-indigo hue (260°) for a professional enterprise feel
- Status colors designed for both light and dark themes with sufficient contrast
- Calendar component required minimal fix: removed invalid `table` classname key for react-day-picker v10 compatibility
